== Description ==

The "Searchmetrics Content Experience WordPress Plugin" builds the bridge between WordPress and Searchmetrics. This opens up completely new possibilities for you as a user of these services - and allows you more efficiency in writing SEO-optimized content.
This powerful plugin helps you save valuable time when preparing and optimizing your texts.


* Keywords: Let the WordPress Editor show you the keywords you have selected in your Searchmetrics Briefing.
* Questions: Besides keywords, our plugin also shows you selected questions that you should answer in your text.
* Content Score: Keep an eye on your content score at all times - even while you are writing. We have placed the score at the top of the editor for this purpose.


== Installation ==

Starting with the Searchmetrics Content Experience WordPress Plugin consists of just one step.


1. Upload the ‘Searchmetrics-Content-Experience-WordPress-Plugin’ folder to the /wp-content/plugins/ directory;
1. Activate the Searchmetrics Content Experience WordPress Plugin through the ‘Plugins’ menu in WordPress;
1. Start our Setup Tutorial above.
